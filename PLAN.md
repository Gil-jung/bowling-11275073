## 1. 목표 (RED 1단계)

**단 하나의 동작**: 10프레임에서 스페어가 나오면, 같은 프레임 안에서
보너스 롤 1개가 추가로 던져지고, 그 롤이 그대로 보너스로 합산된다
(그 이후 더 이상의 프레임/롤은 없다).

## 2. 작성할 실패 테스트 — 그리고 사전 고지

- 파일: `test_bowling.py`
- 테스트: `test_tenth_frame_spare_gets_one_bonus_roll`

```python
def test_tenth_frame_spare_gets_one_bonus_roll():
    game = Game()
    for _ in range(9):
        game.roll(0)
        game.roll(0)   # 1~9프레임: 전부 거터
    game.roll(5)
    game.roll(5)       # 10프레임: 스페어
    game.roll(3)       # 보너스 롤

    # 1~9프레임: 0
    # 10프레임: 10 + 3 = 13
    assert game.score() == 13
```

**사전 고지**: Cycle 4와 동일한 이유로, 이 테스트도 코드 변경 없이 이미 통과할
가능성이 높다 (스페어 보너스 롤 역시 "다음 인덱스의 값"으로 동일하게 처리되기
때문). 실제 실행 결과를 직접 확인해 RED/GREEN 여부와 그 이유를 판단한다.

- 만약 이미 GREEN이면: 프로덕션 코드 변경 없이, 회귀 방지용 확인 테스트로만 커밋한다.
- 만약 실패한다면: 그 실패 원인(예: 프레임 종료 조건 오류)에 따라 GREEN 구현을 진행한다.

## 3. 구현 방향 (필요할 경우에만, GREEN에서 진행)

테스트가 실제로 실패할 경우에만: 10프레임 종료 조건(프레임 카운트 vs. 남은 롤 수)을
